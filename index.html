Welcome to the HTML/CSS Cheatsheet

Begin every HTML document with <html>

The <head> tag is for information about the webpage such as what the title of that tab will be called and where you link things like css stylesheets.

The <link> tag has no closing tag and has 3 parts to it:

  • href, which is the name of the css file being linked,
  • type, the type of file being linked (which is usually css so text/css is what would be filled in),
  • and rel which defines the purpose of the file, which is usually just stylesheet.

The <body> tag is where all of the actual HTML that will show up on the page goes

HTML Tag/CSS Selector Tag Format Tag Description
Image <img> Self closing tag that links an image to the html document. src inside tag should be name of image, alt provides text if browser can't load image. Can set width and height within tag as well.
Title <title> Sets name of website tab to whatever is within the Tags
Div <div> Allows you to divide HTML code into sections. Can assign those sections with class properties to allow stylization to specific aspects of your page.
Nav <nav>, <a> Contains set of links to allow users to navigate your page quicker by jumping to specific sections.
Font Family font-family Sets desired scope to selected font.
Classes .(insert class name) Similar to div but for CSS, allows for specific stylization of specific aspects of HTML doc by assigning a class
Background Image background-image: url(name of image file); Sets background of specified section to whatever image was selected.
Font Size font-size Sets size of text in specified section to desired number (ex: 20px).
